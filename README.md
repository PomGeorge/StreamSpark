# TwitchStreamSpark 🎮🔥

TwitchStreamSpark is a modular Python tool that detects highlight-worthy moments in Twitch streams by analyzing spikes in chat activity. It identifies periods of peak audience engagement using emote bursts, keyword frequency, and message volume.

Whether you're a streamer, editor, or just curious about what moments got the most hype, TwitchStreamSpark helps you find the good stuff—fast.

---

## 🚀 Features

- 🔍 Parses Twitch chat logs into structured, timestamped messages  
- ⚙️ Scores highlight segments based on emotes, keywords, and message density  
- 🧱 Built with modular OOP design: separate components for parsing, scoring, and exporting  
- 🧠 Optimized for fast performance with scalable data structures  
- 📄 Outputs top highlight timestamps to CSV for use in editors or scripts  
- 🛠️ (Planned) FFmpeg integration for automatic video clipping  
- 🌐 (Planned) Optional Flask or React UI for highlight browsing

---
